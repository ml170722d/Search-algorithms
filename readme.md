# Search algorithms

---

#### To run one of algorithms run

```bash
    python ./main.py ./maps/map{id}.txt {agent}
```

* id = 0-4
* agent:
  * Aki (BFS)
  * Jocke (DFS)
  * Draza (Branch & Bound)
  * Bole (A star)

To run all tests at once, use runAll.bat script
